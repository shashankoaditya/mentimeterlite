# Mentimeter Lite (Internal Team Q&A)

A simple Mentimeter-style live Q&A application built using **Streamlit**.  
This app allows internal team members to submit answers to questions during meetings or training sessions.

The application has two modes:

- Presenter Mode
- Participant Mode

---

## Features

- Live question publishing
- Participants can submit answers
- Simple dashboard for viewing responses
- Lightweight and easy to deploy
- Suitable for internal team sessions
